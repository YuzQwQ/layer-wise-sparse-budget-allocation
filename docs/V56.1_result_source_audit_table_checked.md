## Global conventions checked

- Delta convention: `Delta Val = ValLoss(uniform) - ValLoss(method)`, `Delta PPL = PPL_test(uniform) - PPL_test(method)`.
- Positive delta means better than uniform within the same sparse platform, model, dataset, and average budget.
- NSA static schedules use `actual_k = 16.0` and `estimated selected KV access = 1024.0`.
- H2O-style diagnostics use `recent_size = 128`, `avg HH budget = 128`, and `avg retained = 256`.
- Sliding-window diagnostics use `context length = 2048` and `avg window size = 512`.
- Cross-operator diagnostics are inference-time diagnostics on `facebook/opt-350m`, not reproductions or benchmarks.
