# Notice

This repository contains code and experiment utilities for a paper-level empirical study of layer-wise sparse budget allocation.

The NSA implementation is based on the Native Sparse Attention codebase and keeps the original MIT license notice where applicable. Additional experiment scripts, diagnostics, summaries, and plotting utilities were added for the sparse budget allocation study.

This repository does not include trained checkpoints, raw dataset caches, Hugging Face model caches, remote machine logs, or credentials.
